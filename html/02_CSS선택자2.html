        태그에 작성 된 특정 속성을 이용해 선택하는 선택자

        작성법
        
            선택자[속성명="속성값"] { CSS 코드; }

자식 선택자( > )

        지정된 요소 바로 한 단계(1 level,/depth) 하위 요소 선택

        작성법
        
            선택자1 > 선택자2 {css 코드}

            - 선택자1 : 부모 요소( 감싸고 잇는 요소 )
            - 선택자2 : 자식 요소( 하위 요소)
  • 자식1
  • 자식2
  • 자식3
  • 자식4

후손 선택자 (작성법:띄어쓰기)

        지정된 요소의 모든 하위 요소 선택 (n level 이하 모두)

        작성법 
        
            선택자1 선택자2 { CSS 코드; }

            선택자1: 부모(조상) 요소
            선택자2: 후손 요소
